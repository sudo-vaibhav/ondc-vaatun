## Visual Style

### Overall Aesthetic

| Aspect | Specification |
|--------|---------------|
| **Style** | Modern minimalist with subtle 3D elements |
| **Mood** | Optimistic, calm, reassuring |
| **Lighting** | Soft, diffused, warm natural light |

### Photography Style

| Element | Guideline |
|---------|-----------|
| **People** | Diverse Indian demographics, genuine expressions, not overly posed |
| **Settings** | Clean, modern, relatable Indian contexts |
| **Composition** | Rule of thirds, ample negative space for text overlay |
| **Color Treatment** | Slightly desaturated, consistent warm tone |
| **Avoid** | Stock photo clichés, overly corporate, Western-only contexts |

### Illustration Style

| Element | Guideline |
|---------|-----------|
| **Type** | Flat with subtle gradients and soft shadows |
| **Line Weight** | Medium, consistent, rounded ends |
| **Characters** | Simple, friendly, diverse skin tones |
| **Objects** | Recognizable, not overly detailed |
| **Animation-Ready** | Clean shapes suitable for motion |

```
│   ░░░░░░░░░░░░░░░░░░░░░░░░                                 │
│   ░░  TEXT SAFE ZONE   ░░          [Subject/Focus Area]    │
│   ░░  (Left 40-50%)    ░░                                  │
│   ░░░░░░░░░░░░░░░░░░░░░░░░                                 │
│                                                             │
│                                        Keep clear for       │
│                                        visual interest      │
└─────────────────────────────────────────────────────────────┘

Aspect Ratio: 16:9 or 21:9 for cinematic
Text Safe Zone: Left 40-50% of frame
Subject Placement: Right side or rule-of-thirds intersection
```

#### Card Images (Square/4:3)

```
┌─────────────────────┐
│                     │
│    [Subject in      │
│     center or       │
│     lower third]    │
│                     │
│   ─────────────     │
│   Text overlay area │
└─────────────────────┘

Aspect Ratio: 1:1 or 4:3
Bottom Safe Zone: Reserve bottom 25% for text if needed
Subject: Centered or lower third placement
```

#### Icon/Spot Illustrations

```
┌─────────────┐
│             │
│   [Icon]    │
│             │
└─────────────┘

Aspect Ratio: 1:1

```

## Do's and Don'ts

### DO

| Category | Guidelines |
|----------|------------|
| **People** | Use diverse Indian faces; show genuine emotions; include various age groups |
| **Settings** | Feature recognizable Indian contexts; use modern, aspirational but relatable environments |
| **Colors** | Stay within brand palette; use teal for health, blue for motor consistently |
| **Composition** | Leave space for text; use consistent lighting; maintain visual hierarchy |
| **Quality** | Generate at highest resolution; ensure sharp details; maintain consistency |

### DON'T

| Category | Avoid |
|----------|-------|
| **People** | Overly Western contexts; fake/plastic expressions; stereotypical poses |
| **Settings** | Dated interiors; cluttered backgrounds; identifiable competitor branding |
| **Colors** | Neon or garish colors; inconsistent color grading between images |
| **Composition** | Centered subjects with no text space; busy/cluttered frames |
| **Clichés** | Handshake images; pointing at charts; thumbs up; excessive use of shields |

### Specific Avoidances

```
AVOID in Health Images:
- Sad/worried expressions (even in "before" contexts)
- Hospital beds with patients looking ill
- Overly clinical/sterile environments
- Injection/needle imagery (can cause anxiety)

AVOID in Motor Images:
- Accident scenes or damaged vehicles
- Reckless driving depictions
- Overly luxurious cars (not relatable)
- Traffic jams or frustrating scenarios
```

## AI Generation Parameters

### Recommended Settings

| Tool | Settings |
|------|----------|
| **Midjourney** | `--ar 16:9 --style raw --stylize 100` for photos; `--stylize 250` for illustrations |
| **DALL-E 3** | Quality: HD; Style: Natural (for photos), Vivid (for illustrations) |
| **Stable Diffusion** | Steps: 30-50; CFG Scale: 7-8; Sampler: DPM++ 2M Karras |

### Prompt Structure

```
[Subject] + [Context/Setting] + [Lighting] + [Mood] + [Style] + [Technical]

Example:
"Indian family of four" + "in modern living room" + "warm natural lighting" +
"happy, relaxed atmosphere" + "lifestyle photography" + "8k, shallow DOF"
```

### Negative Prompts (Stable Diffusion)

```
Standard Negative:
ugly, blurry, low quality, watermark, text, logo, western faces only,
stock photo watermark, oversaturated, cartoon (unless intended),
deformed hands, extra limbs

Health-Specific Negative:
blood, injury, medical gore, sad expressions, hospital gown

Motor-Specific Negative:
car crash, accident, damage, scratches, dirty vehicle, traffic jam
```

## Quality Checklist

Before using any generated image:

- [ ] Matches brand color palette (teal/blue/slate)
- [ ] Appropriate for Indian context
- [ ] No anatomical errors (hands, faces)
- [ ] No text/watermarks in image
- [ ] Sufficient resolution (min 2x display size)
- [ ] Leaves space for text overlay if needed
- [ ] Consistent lighting/mood with other assets
- [ ] No competitor branding visible
- [ ] Passes emotional tone check (positive, trustworthy)
- [ ] Accessible (sufficient contrast, clear subjects)
